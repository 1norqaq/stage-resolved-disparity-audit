# Data Use Terms

## Governed empirical data

The governed empirical data are not included in this repository. They remain subject to the data controller's access agreement and are not licensed for redistribution.

## Synthetic data

Synthetic data generated by `scripts/make_synthetic_data.py` are released under CC0 1.0 Universal. The synthetic rows contain no real applicant records and are intended only for code-path testing.

## Aggregate outputs

Aggregate tables and figures produced from governed data should be reviewed for disclosure risk before public release. The repository's MIT License applies to code, not to protected empirical data.
